Docs : API : Get App

The Get App API lets you retrieve the name, id, and creation date for the app and all related users by providing the ID of the app. The app ID can be obtained from the Apps page.

    
      curl 'https://v1.userbase.com/v1/admin/apps/$APP_ID' \
        -XGET \
        -H 'Authorization: Bearer $ACCESS_TOKEN'

Parameters

  • APP_ID - The app ID to retrieve.
  • ACCESS_TOKEN - Your access token.

Result

  • appName [string] - The app's name.
  • appId [string] - The app's unique identifier.
  • creationDate [string] - The timestamp when the user was created.
  • users [array] - A list of user objects.
  • users.username [string] - The user's username.
  • users.userId [string] - The user's unique identifier.
  • users.appId [string] - The app id which the user belogs to.
  • users.creationDate [string] - The timestamp when the user was created.
